## 🚀 Technical Implementation

### 🔧 Dual Format Decoders
Each affected device now supports both formats:

```python
def decode_htv213frf_valve(raw: str) -> dict:
    if raw.startswith("11#"):
        return _decode_htv213frf_hex(raw)
    elif "," in raw and (";" in raw or "|" in raw):
        return _decode_htv213frf_ascii(raw)
    else:
        raise ValueError(f"Unexpected payload format: {raw}")
```

### 🔍 ASCII Valve Parsing
```python
# Parse: 1,-84,1;0,149,0,0,0,0|0,6,0,0,0,0
# Structure: [flags],[rssi],[flags];[zone1_data]|[zone2_data]

header_part, zone_part = raw.split(";", 1)
header_parts = header_part.split(",")  # 1,-84,1
rssi_raw = int(header_parts[1])        # -84 dBm
zone_sections = zone_part.split("|")   # Zone data sections
```

### 🔍 ASCII Sensor Parsing
```python
# Parse: 1,-73,1;694,70,G=292478
# Structure: [flags],[rssi],[flags];[temp_raw],[moisture],[lux_data]

header_part, sensor_part = raw.split(";", 1)
sensor_parts = sensor_part.split(",")
temp_raw = int(sensor_parts[0])          # 694 → 69.4°C
moisture = int(sensor_parts[1])        # 70%
lux_data = sensor_parts[2]              # G=292478 → 29247.8 lux
```
